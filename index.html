My Projects



RISC-V Pipeline Processor
SystemVerilog, RISC-V, VCS, RTL design/verification

Designed and implemented a 5-stage RISC-V RV32I pipeline processor in SystemVerilog, optimizing performance with forwarding, backpressure-based stall, and global stall models to resolve hazards. Achieved 500 MHz clock frequency and 13,226 μm² chip area. Verified functionality using VCS.




Image Comparator Circuit
Virtuoso, HSPICE

Designed a 4-bit comparator with reliable operation across all corners, achieving 1.82 GHz performance. Led a four-person team through the full design process in Virtuoso, optimizing transistor sizing and layout to achieve a compact 58.52 × 46.01 μm² area.



Autonomous Driving Software for GEM Electric Vehicle
ROS, RVis, Python, Ubuntu, Computer Vision

Developed an autonomous driving system with lane detection using front camera signals and control integration. Implemented Gradient Filter, Color Filter, and YOLOv2 for lane detection, Pure Pursuit for steering, and PID for acceleration, enabling navigation through stop signs, pedestrians, and roadwork.



Snake Game on FPGA Board
Verilog, FPGA

Designed and implemented a Snake game in Verilog, featuring VGA display, keyboard controls, and real-time scoring. Integrated stereo background music and deployed the game on an FPGA board.


ROM Peripheral Circuit
HSPICE, Composer, Laker

Developed a 10-bit addressable ROM circuit for accurate data retrieval based on input addresses. Optimized transistor sizing and layout design for functionality across all corners, achieving a nearly rectangular 96.3 × 238.3 μm² layout.





Multi-Bit Storage Using HfZrOx Ferroelectric Films

Investigated HfZrOx films for multi-bit storage, optimizing the capacitor manufacturing process by switching from PMA to PDA, achieving 10⁸ cycle reliability. Leveraged ferroelectric properties to create 8 distinct states, enabling 3-bit storage. Awarded the 1H Research Assistant Scholarship by TSMC in 2022.
